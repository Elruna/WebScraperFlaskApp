Welcome to the Ceneo Product Review Extractor

This web application extracts complete descriptive parameters, star reviews, and feature details from Ceneo.pl entries using product reference index keys.

Purpose of the Application

Automate user review harvesting and present clean statistical models for customer feedback analytics.

Description of Operation

Provide an identifier string, and the system loops across the page layout structures, structures the information fields, saves records into local storage, and builds visualization charts.

Libraries Used

  • Flask: Serves the backend application server components.
  • BeautifulSoup4 & Requests: Web parsing core automation services.
  • Matplotlib: Compiles charts on the background workers.

Applied Approaches

Built starting from flat linear structures up to layered Object-Oriented layouts employing explicit entity relationship logic abstractions.
